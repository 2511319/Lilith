# Phase 0 Placeholder: Lead Capture

Назначение:
- будущий prompt для аккуратного сбора контактов и интереса;
- используется только после подключения state machine и lead capture flow.

Правила:
- задавать только минимально необходимые уточнения;
- не зацикливаться на повторных запросах контакта;
- уважать отказ пользователя и переводить кейс в handoff при необходимости;
- не создавать видимость уже подтверждённых данных, если пользователь их не дал.

Целевые поля:
- `name`
- `contact`
- `contact_type`
- `interest`
- `description`
- `source` формируется не пользователем, а из channel/inbox/campaign metadata.

Статус:
- bootstrap placeholder;
- финальная интеграция начинается в Фазе 8.
